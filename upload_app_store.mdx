<Info>
  **Prerequisite** You should have an Apple Developer account to submit apps to the App Store.
</Info>

Follow these steps to submit your app to the App Store:

1. Create an Expo account:
   - Go to [Expo.dev](https://expo.dev/) and sign up for a free account.
   - This allows you to upload 15 builds to the App Store every month.

2. Set up Expo access token:
   - In Expo, go to Account settings > Access tokens.
   - Click on "Add robot" button and select the developer role.
   - Click "Create robot" and copy the generated token.
   - In your project settings, go to the Expo section.
   - Paste the token into the "Robot token" field and save.

3. Generate App Store Connect API key:
   - Go to [App Store Connect](https://appstoreconnect.apple.com/).
   - Navigate to Users and Access > Keys.
   - Click the "+" button to generate a new API key.
   - Note down the Issued ID and Key ID.
   - Download the .p8 file.

4. Retrieve Team ID:
   - Go to [developer.apple.com](https://developer.apple.com/).
   - Find your Team ID in the Membership details.

5. Upload App Store information:
   - In your project settings, go to the App Store section.
   - Enter the Issued ID, Key ID, and Team ID.
   - Upload the .p8 file you downloaded.
   - Save the changes.

6. Set bundle identifier:
   - Go to Settings > App Info.
   - Set your unique bundle identifier.

7. Deploy to App Store:
   - Go to the Deployments section.
   - Click the "+" button in the top right to start a new deployment.
   - Follow the prompts to complete the deployment process.

8. Create app in App Store Connect:
   - Go back to [App Store Connect](https://appstoreconnect.apple.com/).
   - Create a new app with the same bundle identifier you set earlier.

9. Access build in TestFlight:
   - Once the build is complete, go to the TestFlight tab in App Store Connect.
   - You should find your pushed build accessible there.

<Tip>
  Make sure your app meets all of Apple's guidelines before submitting for review to increase your chances of approval.
</Tip>

## Next Steps

After submitting your build to TestFlight, you can invite testers and prepare for final submission to the App Store. Refer to Apple's documentation for more information on the app review process and requirements.
