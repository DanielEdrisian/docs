<Info>
  **Prerequisite** You should have a Google account to create a Firebase project.
</Info>

Follow these steps to set up Firebase for your project:

1. Create a Firebase project:
   - Go to the [Firebase Console](https://console.firebase.google.com/).
   - Click on "Add project" or "Create a project".
   - Enter a project name and follow the prompts to set up your project.

2. Generate a configuration for a Firebase web app:
   - In your Firebase project console, click on the gear icon next to "Project Overview" and select "Project settings".
   - In the "Your apps" section, click the "+" icon to add a web app.
   - Register your app by providing a name (e.g., "My Web App").
   - Firebase will generate a configuration object. It will look similar to this:

```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
  projectId: "YOUR_PROJECT_ID",
  storageBucket: "YOUR_PROJECT_ID.appspot.com",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

3. Copy the Firebase configuration:
   - Select and copy the entire `firebaseConfig` object.

4. Add the configuration to your project:
   - Go to your project's settings page.
   - Navigate to the Firebase settings section.
   - Paste the copied configuration into the provided input field.
   - Click "Save" to store your Firebase configuration.

<Tip>
  Keep your Firebase configuration secure and don't share it publicly, as it contains sensitive information about your project.
</Tip>

## Next Steps

After setting up Firebase, you can start using Firebase services in your project. Refer to the Firebase documentation for more information on how to use specific features like Authentication, Firestore, or Cloud Functions.
